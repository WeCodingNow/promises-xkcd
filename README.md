# Promises example

Простые cli-инструменты для получения мета-данных про комиксы XKCD

## Requirements

- nodejs

## Usage

Заходим в репозиторий
```bash
npm install
```

Далее:

```bash
npm run xkcd-getter 1
```

```bash
npm run xkcd-getter-multi 1 2 3
```

```bash
npm run xkcd-getter-async 1
```
